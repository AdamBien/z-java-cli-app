# z-java-cli-app
A minimalistic template for zero-dependency Java 25 applications — multi-file CLI apps packaged as executable JARs with zb (Zero Dependencies Builder), no Maven or Gradle.

Scaffold and maintain this kind of project with the [`/java-cli-app`](https://airails.dev) skill, build it with [`/zb`](https://airails.dev), and test it with [`/zunit`](https://airails.dev).

See [airails.dev](https://airails.dev) for more.
